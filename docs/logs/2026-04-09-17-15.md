## 关键变更说明

### 1. 文档存储系统（DocumentStore）

#### Document 数据模型
```typescript
interface Document {
  id: string
  title: string
  content: string // JSON 格式的编辑器内容
  blocks: DocumentBlock[] // 文档中的隐式 Block
  metadata: {
    createdAt: Date
    updatedAt: Date
  }
}
```

#### DocumentBlock 数据模型
```typescript
interface DocumentBlock {
  id: string // Block ID
  nodeType: string // paragraph, heading, listItem 等
  content: string // 纯文本内容
  position: number // 在文档中的位置
  links: string[] // 该段落中的链接
}
```

**核心功能**:
- 文档 CRUD 操作
- 隐式 Block 自动提取
- 链接关系自动维护
- 当前文档管理

### 2. 隐式 Block 系统

#### 自动提取算法
```typescript
extractImplicitBlocks(editorJSON: any): DocumentBlock[]
```

**工作原理**:
1. 遍历编辑器 JSON 结构
2. 识别块级节点（paragraph, heading, listItem）
3. 提取文本内容和链接
4. 分配唯一 ID 和位置索引
5. 生成 DocumentBlock 数组

**支持的节点类型**:
- `paragraph` - 段落（📝）
- `heading` - 标题（📌）
- `listItem` - 列表项（•）

#### 文本内容提取
```typescript
private extractTextContent(node: any): string
```
- 递归遍历节点树
- 提取所有文本节点
- 拼接为完整字符串

#### 链接提取
```typescript
private extractLinks(node: any): string[]
```
- 查找 `blockLink` 节点
- 提取 `blockId` 属性
- 返回链接 ID 数组

### 3. 链接关系自动维护

#### 更新链接关系
```typescript
private async updateLinkRelations(
  documentId: string,
  oldBlocks: DocumentBlock[],
  newBlocks: DocumentBlock[]
): Promise<void>
```

**工作流程**:
1. 收集旧文档的所有链接
2. 收集新文档的所有链接
3. 计算差异（新增 vs 删除）
4. 调用 `blockStore.addLink()` 添加新链接
5. 调用 `blockStore.removeLink()` 移除旧链接

**自动维护时机**:
- 用户编辑文档时（1秒防抖）
- 插入新链接时（立即）
- 删除链接时（自动检测）

#### 即时链接添加
```typescript
// 在 handleSelectSuggestion 中
if (suggestionType === 'link') {
  // 插入链接
  editor.chain().focus().insertContent({
    type: 'blockLink',
    attrs: { blockId: item.id, blockTitle: item.title }
  }).run()

  // 立即建立双向关系
  await blockStore.addLink(currentDocumentId, item.id)
}
```

### 4. Editor 组件增强

#### 文档初始化
```typescript
useEffect(() => {
  const initDocument = async () => {
    await documentStore.init()
    
    // 创建或加载默认文档
    const docs = await documentStore.getAllDocuments()
    let doc = docs.length === 0 
      ? await documentStore.createDocument('我的第一篇文档')
      : docs[0]
    
    setCurrentDocumentId(doc.id)
    documentStore.setCurrentDocument(doc.id)
    
    // 加载文档内容到编辑器
    if (doc.content) {
      editor.commands.setContent(JSON.parse(doc.content))
    }
  }
  
  if (editor) initDocument()
}, [editor])
```

#### 文档自动保存（防抖）
```typescript
onUpdate: ({ editor }) => {
  // ... 自动补全逻辑 ...
  
  // 延迟更新文档（防抖）
  if (updateTimeoutRef.current) {
    clearTimeout(updateTimeoutRef.current)
  }
  updateTimeoutRef.current = setTimeout(() => {
    handleDocumentUpdate(editor)
  }, 1000) // 1秒后更新
}
```

#### 文档更新处理
```typescript
const handleDocumentUpdate = async (editor: TiptapEditor) => {
  if (!currentDocumentId) return

  try {
    const editorJSON = editor.getJSON()
    await documentStore.updateDocumentBlocks(currentDocumentId, editorJSON)
  } catch (error) {
    console.error('Failed to update document blocks:', error)
  }
}
```
