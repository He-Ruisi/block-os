### 文件变更
- 无文件变更

### 关键说明
- **原因**：bun 安装在 `~/.bun/bin/bun`，但未加入 shell PATH
- **方案 1（推荐）**：永久修复，将 bun 加入 PATH：
  ```bash
  echo 'export PATH="$HOME/.bun/bin:$PATH"' >> ~/.zshrc
  source ~/.zshrc
  ```
- **方案 2**：每次使用完整路径 `~/.bun/bin/bun run dev`
- steering 文件 `bun-defaults.md` 中已记录完整路径备用
