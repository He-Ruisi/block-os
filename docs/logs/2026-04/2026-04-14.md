## 15:30 - 修复左侧边栏点击不显示的严重 Bug
### 问题描述
- 点击 ActivityBar 上方的项目、大纲等按钮后，左侧边栏快速闪一下但不显示
- 用户无法正常呼出左侧边栏，核心功能不可用
- 影响范围：桌面/平板/手机所有设备

### 根本原因
1. **逻辑冲突**：`handleSidebarViewChange` 使用 `toggleSidebar()` 切换状态，导致异步状态更新不一致
2. **组件提前返回**：`Sidebar` 组件在 `collapsed=true` 时直接返回 `null`，导致重新挂载闪烁

### 修复方案
1. **App.tsx**：将 `toggleSidebar()` 改为直接设置 `setSidebarCollapsed(true/false)`，状态明确不依赖切换
2. **Sidebar.tsx**：移除 `if (collapsed) return null`，改用 `style={{ display: collapsed ? 'none' : 'flex' }}` 控制显示

### 文件变更
- `src/App.tsx` - 修复 `handleSidebarViewChange` 逻辑
- `src/components/layout/Sidebar.tsx` - 移除提前返回，使用 CSS 控制显示
- `docs/tests/bug-fix-sidebar-flash.md` - 新建 Bug 修复文档

### 验收标准
- ✅ 点击 ActivityBar 图标，左侧边栏正常展开
- ✅ 点击已激活的图标，左侧边栏正常折叠
- ✅ 切换不同视图，左侧边栏正常切换内容
- ✅ 无闪烁，无延迟，交互流畅
- ✅ TypeScript 类型检查通过

### 关键说明
- 这是一个 P0 级别的严重 Bug，阻塞核心功能使用
- 修复后需要在所有设备（桌面/平板/手机）上测试验证
- 经验教训：避免使用 toggle 函数，直接设置状态更可靠；使用 CSS 控制显示避免重新挂载

## 16:00 - 修复 Block 发布版本和引用记录问题
### 问题描述
1. **发布版本无响应**：编辑区域的 block 编辑后点击"发布版本"按钮，版本数不增加
2. **引用记录为 0**：block 空间的引用记录一直显示为 0，即使已经发布了多个版本

### 根本原因
1. **releaseVersion 未更新**：发布成功后虽然创建了新版本，但节点的 `releaseVersion` 属性没有被更新
2. **usage 记录条件错误**：使用了 `if (currentReleaseVersion !== release.version)` 条件判断，导致首次发布时不记录 usage

### 修复方案
1. **移除条件判断**：删除 `if (currentReleaseVersion !== release.version)` 条件，每次发布都调用 `recordBlockUsage`
2. **立即更新节点**：在调用 `recordBlockUsage` 之前先更新节点的 `releaseVersion` 和 `sourceLabel` 属性
3. **触发更新事件**：通过 `window.dispatchEvent(new Event('blockUpdated'))` 通知其他组件刷新

### 文件变更
- `src/editor/extensions/sourceBlock.ts` - 修复发布逻辑
- `docs/tests/block-release-usage-test.md` - 新建测试文档

### 修复前的代码
```typescript
// ❌ 条件判断导致不记录 usage
if (currentReleaseVersion !== release.version) {
  await recordBlockUsage(blockId, release.version)
}
```

### 修复后的代码
```typescript
// ✅ 立即更新节点属性
if (typeof getPos === 'function') {
  const pos = getPos()
  editor.chain().focus().command(({ tr }: { tr: any }) => {
    tr.setNodeMarkup(pos, undefined, {
      ...nodeAttrs,
      releaseVersion: release.version,
      sourceLabel: `📦 v${release.version} · ${release.title}`,
    })
    return true
  }).run()
}

// ✅ 每次发布都记录 usage
await recordBlockUsage(blockId, release.version)
```
