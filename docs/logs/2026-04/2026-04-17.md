**决策**：创建轻量级 Hook（可选，P1 优先级）

**Hook 配置**：
```json
{
  "name": "shell-component-detector",
  "when": {
    "type": "fileEdited",
    "patterns": ["src/features/**/components/**/*View.tsx"]
  },
  "then": {
    "type": "askAgent",
    "prompt": "检查是否有重复 UI 模式（className >8 个类、重复 UI 组合）"
  }
}
```

**触发条件**：
1. 重复 UI 模式（≥2 个 View）
2. 长 className（>8 个 Tailwind 类）
3. 复杂组合（多个 Shadcn UI 组合）
4. v0 导出（长 className 代码）
