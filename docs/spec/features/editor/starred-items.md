# 置顶功能（Starred Items）

**状态**: 已完成  
**优先级**: P1  
**完成日期**: 2026-04-13

## 概述

为 BlockOS 添加置顶功能，允许用户将常用的项目和文档置顶到侧边栏，方便快速访问。

## 功能特性

### 1. 置顶视图

- **位置**: 侧边栏第三个图标（星标图标）
- **显示内容**: 用户置顶的项目和文档列表
- **排序**: 按置顶时间倒序（最新置顶的在最上面）

### 2. 置顶操作

- **添加置顶**: 在项目或文档上点击星标图标
- **取消置顶**: 在置顶列表中点击星标图标，或在原位置再次点击
- **支持类型**: 项目（Project）和文档（Document）

### 3. 数据持久化

- **存储位置**: localStorage (`blockos-starred-items`)
- **数据结构**:
  ```typescript
  interface StarredItem {
    id: string
    type: 'project' | 'document'
    name: string
    projectId?: string // 如果是文档，记录所属项目
    starredAt: Date
  }
  ```

### 4. 交互行为

- **点击项目**: 切换到该项目视图
- **点击文档**: 打开该文档
- **悬停显示**: 显示取消置顶按钮
- **空状态**: 显示引导提示

## 技术实现

### 组件结构

```
StarredView.tsx
├── 加载置顶列表（localStorage）
├── 监听全局置顶事件（toggleStar）
├── 渲染置顶项目列表
└── 处理点击和取消置顶
```

### 全局事件

```typescript
// 添加/取消置顶
window.dispatchEvent(new CustomEvent('toggleStar', {
  detail: {
    id: string,
    type: 'project' | 'document',
    name: string,
    projectId?: string,
  },
}))
```

### 数据流

```
用户点击星标
    ↓
触发 toggleStar 事件
    ↓
StarredView 监听事件
    ↓
更新 localStorage
    ↓
重新渲染列表
```
