# Supabase 认证与数据同步

## 功能概述
集成 Supabase 后端，实现用户认证和云端数据存储。

## 认证方案
- 用户名+密码注册/登录，无需邮箱验证
- 用户名转为 `username@blockos.local` 伪邮箱格式
- 密码最少 6 字符，用户名最少 3 字符

## 数据表
- `projects` - 项目数据
- `documents` - 文档数据
- `blocks` - 卡片/Block 数据
- 所有表启用 RLS，用户只能访问自己的数据

## 同步策略
- 本地 IndexedDB 作为主存储（离线可用）
- Supabase 作为云端备份和跨设备同步
- 本地操作后异步同步到云端
