# BlockOS 产品概述

BlockOS 是一个写作优先、AI 原生的个人知识操作系统（Web 应用）。

## 核心理念
- 用户通过 AI 对话探索灵感，通过编辑器产出内容
- Block 是最小知识单元，可被引用、链接、派生
- 内容（Content）、样式（Style）、模板（Template）三层解耦

## 主要功能
- TipTap 富文本编辑器，支持 `[[双向链接]]` 和 `((块引用))`
- AI 对话（MiMo API），流式响应，AI 回复自动创建隐式 Block
- Block 空间：显式 Block 管理、搜索、标签过滤
- Block 派生系统：同一 Block 在不同文档中可产生派生版本
- 多项目/多文档工作区，标签页管理
- IndexedDB 本地存储，Supabase 可选云同步

## 用户语言
项目 UI 和文档均使用中文。代码注释中文为主。
