# AI 高级功能说明

## 功能更新

### 1. 回复内容分离 ✨

AI 的回复现在分为两部分：
- **简短回复**: 直接回复用户的问题或确认
- **编辑器内容**: 用于写入编辑器的实际内容

#### 使用方法

在提问时，明确告诉 AI 需要写入编辑器的内容，例如：
- "帮我写一段关于产品定位的描述"
- "续写第三节的内容"
- "扩展这个观点并写入编辑器"

AI 会自动识别并分离回复：
```
简短回复：好的，我已经为你生成了产品定位的描述。

---CONTENT---
【编辑器内容】
产品定位是...（详细内容）
```

#### 内容预览

- 简短回复显示在对话气泡中
- 编辑器内容显示在预览框中（带 📝 标识）
- 点击"写入编辑器"按钮插入预览框中的内容

### 2. 系统提示词设置 ⚙️

#### 打开设置

点击右上角的 ⚙️ 按钮打开设置面板

#### 自定义系统提示词

系统提示词决定了 AI 的行为和回复风格，你可以：
- 设置 AI 的角色和专业领域
- 定义回复的格式和风格
- 添加特定的约束条件

#### 示例提示词

**写作助手**:
```
你是一位专业的写作助手，擅长帮助用户组织思路和撰写文档。
回复时保持简洁专业，提供的内容要结构清晰、逻辑严密。
```

**技术文档专家**:
```
你是一位技术文档专家，擅长撰写技术文档和 API 文档。
使用准确的技术术语，提供代码示例时要确保可运行。
```

**创意写作**:
```
你是一位创意写作导师，擅长激发灵感和提供创意建议。
回复时要富有想象力，鼓励用户探索不同的写作角度。
```

#### 保存设置

- 点击"保存"按钮应用新的系统提示词
- 点击"取消"放弃修改
- 设置会立即生效，影响后续的所有对话

### 3. 独立滚动区域 🔄

#### 编辑器区域
- 编辑器有独立的滚动条
- 不受 AI 对话区域影响
- 可以自由上下滚动查看内容

#### AI 对话区域
- AI 对话有独立的滚动条
- 新消息自动滚动到底部
- 可以向上查看历史对话

## 最佳实践

### 1. 结构化提问

明确告诉 AI 你的需求：
```
帮我写一段关于【主题】的内容，要求：
1. 字数约 200 字
2. 包含 3 个要点
3. 语气专业正式
```

### 2. 迭代优化

利用对话历史进行迭代：
```
用户：帮我写产品定位
AI：【生成内容】
用户：再简洁一些，突出核心竞争力
AI：【优化后的内容】
```

### 3. 内容预览

在写入编辑器前：
- 查看预览框中的内容
- 确认内容符合预期
- 再点击"写入编辑器"

### 4. 系统提示词调优

根据使用场景调整系统提示词：
- 写技术文档时使用技术专家提示词
- 写创意内容时使用创意写作提示词
- 写商业文档时使用商业写作提示词

## 常见问题

### Q: 如何让 AI 只回复不写入编辑器？
A: 在提问时不要提及"写入"、"生成内容"等关键词，AI 会只进行对话回复。

### Q: 内容分离不生效怎么办？
A: 确保在系统提示词中包含了内容分离的说明，或者在提问时明确要求"写入编辑器"。

### Q: 可以修改内容分离的格式吗？
A: 可以，在系统提示词中自定义分隔符和格式。

### Q: 系统提示词会影响已有对话吗？
A: 不会，只影响保存后的新对话。已有对话使用的是发送时的系统提示词。
