# BlockOS 功能需求文档

本目录按功能模块组织所有需求文档，便于查找和维护。

## 目录结构

```
features/
├── editor/          # 编辑器相关功能
├── ai/              # AI 对话和智能功能
├── block-system/    # Block 系统和知识管理
└── storage/         # 存储和同步功能
```

- 基础 Markdown 支持
- 工具栏操作

**规划中**:
- 高级 Markdown 语法
- 自定义快捷键
- 主题切换
