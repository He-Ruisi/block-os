# 多项目工作区功能需求

## 功能概述

实现多项目管理和多标签页编辑器，支持左侧边栏、项目切换、全屏模式和可调整布局。

## 核心功能

### 1. 左侧边栏（可呼出/收起）

#### 功能描述
- 默认显示，可通过按钮收起/展开
- 收起时显示图标栏，展开时显示完整内容
- 支持快捷键切换（Cmd/Ctrl + B）

#### UI 设计
```
┌─────────────────────────────────────┐
│ [☰] BlockOS                         │
├─────────────────────────────────────┤
│ 📅 今日                              │
│ 📁 项目                              │
│   ├─ 项目 1                         │
│   ├─ 项目 2                         │
│   └─ [+ 新建项目]                   │
├─────────────────────────────────────┤
│ ⚙️ 设置                              │
└─────────────────────────────────────┘
```

#### 交互行为
- 点击 `☰` 按钮：收起/展开侧边栏
- 点击"今日"：切换到今日文档
- 点击项目名：切换到该项目
- 点击"+ 新建项目"：弹出对话框创建新项目

### 2. 项目管理

#### 数据模型
```typescript
interface Project {
  id: string
  name: string
  description?: string
  documents: string[] // 文档 ID 列表
  metadata: {
    createdAt: Date
    updatedAt: Date
    color?: string // 项目颜色标记
    icon?: string  // 项目图标
  }
}
```

#### 功能列表
- ✅ 创建项目（名称、描述、颜色）
- ✅ 重命名项目
- ✅ 删除项目（需确认）
- ✅ 项目列表显示
- ✅ 项目切换

#### 特殊项目
- **今日**：特殊项目，显示今天创建/修改的所有文档
- **所有文档**：显示所有文档（可选）

### 3. 多标签页编辑器

#### 功能描述
- 每个项目/文档在独立标签页中打开
- 支持多个标签页同时打开
- 标签页可关闭、切换、重排序

#### UI 设计
```
┌─────────────────────────────────────────────────────┐
│ [今日] [项目1] [项目2] [+]              [全屏] [⚙️] │
├─────────────────────────────────────────────────────┤
│                                                     │
│              编辑器内容区域                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### 标签页功能
- 显示文档/项目名称
- 未保存状态标记（圆点）
- 关闭按钮（×）
- 拖拽重排序
- 右键菜单（关闭、关闭其他、关闭所有）

#### 默认行为
- 启动时自动打开"今日"标签页
- 点击项目时在新标签页打开
- 最多同时打开 10 个标签页（可配置）

### 4. 全屏模式

#### 功能描述
- 点击全屏按钮：隐藏左侧边栏和右侧面板，编辑器占满整个窗口
- 再次点击：恢复原布局
- 快捷键：F11 或 Cmd/Ctrl + Shift + F

#### 全屏状态
- 隐藏左侧边栏
- 隐藏右侧 AI 面板
- 编辑器占满 100% 宽度
- 顶部保留标签栏和全屏按钮
- 鼠标移到边缘时显示半透明侧边栏（可选）

### 5. 可调整布局（编辑器 ↔ AI 面板）

#### 功能描述
- 编辑器和 AI 面板之间有可拖拽的分隔条
- 拖拽分隔条调整两侧宽度比例
- 记住用户的布局偏好

#### 默认比例
- 编辑器：60%
- AI 面板：40%

#### 最小/最大宽度
- 编辑器最小：400px
- AI 面板最小：300px
- 编辑器最大：80%
- AI 面板最大：60%

#### 交互行为
- 鼠标悬停在分隔条上：显示拖拽光标（↔）
- 拖拽时：实时更新宽度
- 双击分隔条：恢复默认比例（60/40）

## 技术实现

### 1. 组件结构

```
App
├─ Sidebar (左侧边栏)
│  ├─ SidebarHeader (头部 + 收起按钮)
│  ├─ TodaySection (今日)
│  ├─ ProjectList (项目列表)
│  │  ├─ ProjectItem (项目项)
│  │  └─ NewProjectButton (新建按钮)
│  └─ SidebarFooter (底部设置)
├─ EditorArea (编辑器区域)
│  ├─ TabBar (标签栏)
│  │  ├─ Tab (标签页)
│  │  ├─ NewTabButton (新建标签)
│  │  └─ FullscreenButton (全屏按钮)
│  └─ EditorContent (编辑器内容)
├─ ResizeHandle (可拖拽分隔条)
└─ RightPanel (右侧 AI 面板)
```

### 2. 状态管理

```typescript
// 全局状态
interface AppState {
  // 侧边栏
  sidebarCollapsed: boolean
  
  // 项目
  projects: Project[]
  currentProjectId: string | null
  
  // 标签页
  tabs: Tab[]
  activeTabId: string
  
  // 布局
  isFullscreen: boolean
  editorWidth: number // 百分比
  
  // 用户偏好
  preferences: {
    defaultEditorWidth: number
    maxTabs: number
    autoSaveInterval: number
  }
}

interface Tab {
  id: string
  type: 'today' | 'project' | 'document'
  projectId?: string
  documentId?: string
  title: string
  isDirty: boolean // 是否有未保存的更改
}
```

### 3. 数据持久化

使用 IndexedDB 存储：
- 项目列表
- 标签页状态
- 布局偏好
- 侧边栏状态

### 4. 快捷键

| 快捷键 | 功能 |
|--------|------|
| Cmd/Ctrl + B | 切换侧边栏 |
| Cmd/Ctrl + T | 新建标签页 |
| Cmd/Ctrl + W | 关闭当前标签页 |
| Cmd/Ctrl + Tab | 切换到下一个标签页 |
| Cmd/Ctrl + Shift + Tab | 切换到上一个标签页 |
| Cmd/Ctrl + 1-9 | 切换到第 N 个标签页 |
| F11 / Cmd+Ctrl+F | 全屏模式 |

## UI/UX 细节

### 1. 动画效果
- 侧边栏展开/收起：300ms ease-in-out
- 标签页切换：200ms fade
- 全屏切换：400ms ease
- 分隔条拖拽：实时无延迟

### 2. 视觉反馈
- 悬停状态：背景色变化
- 激活状态：高亮边框
- 拖拽状态：半透明预览
- 加载状态：骨架屏

### 3. 响应式设计
- 窗口宽度 < 768px：自动隐藏侧边栏
- 窗口宽度 < 1024px：AI 面板最小化
- 窗口宽度 < 600px：移动端布局（单列）
