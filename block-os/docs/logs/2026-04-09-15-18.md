## 关键变更说明

### Phase 2.5 需求文档

#### 功能范围
1. **选中文字发送给 AI**
   - 右键菜单"发送给 AI"
   - 快捷键 `Cmd/Ctrl + Shift + A`
   - 选中内容作为上下文传递

2. **AI 回复捕获为 Block**
   - AI 消息下方"捕获为 Block"按钮
   - 捕获对话框（标题、标签、预览）
   - 保存到 IndexedDB

3. **Block 空间基础 UI**
   - 右侧面板新增"Block 空间"标签页
   - Block 列表展示
   - 搜索和标签过滤

4. **IndexedDB 存储**
   - 数据库: `blockos-db`
   - Store: `blocks`
   - 索引: tags, createdAt, type

#### 数据模型
```typescript
interface Block {
  id: string
  content: string
  type: 'text' | 'ai-generated'
  source: {
    type: 'editor' | 'ai'
    documentId?: string
    aiMessageId?: string
    capturedAt: Date
  }
  metadata: {
    title?: string
    tags: string[]
    createdAt: Date
    updatedAt: Date
  }
}
```

#### UI 设计
- **编辑器右键菜单**: 新增"发送给 AI"选项
- **AI 消息按钮**: [↗ 写入编辑器] [◆ 捕获为Block]
- **捕获对话框**: 标题、标签、预览、取消/捕获
- **Block 空间面板**: 搜索、标签过滤、Block 卡片列表

#### 技术实现
- **BlockStore 类**: 封装 IndexedDB 操作
- **编辑器选中处理**: getSelectedText + sendSelectionToAI
- **Block 捕获逻辑**: captureBlock + UUID 生成
- **标签页切换**: chat | blocks | session

#### 验收标准
- 功能验收: 8 项核心功能
- 性能验收: 加载 < 100ms, 搜索 < 50ms
- 用户体验验收: 流畅、自然、清晰、便捷
