## 关键变更说明

### 1. Block 数据模型扩展

#### 新增 links 字段
```typescript
interface Block {
  // ... 原有字段
  links?: {
    outgoing: string[]  // 引用的其他 blocks
    incoming: string[]  // 被哪些 blocks 引用
  }
}
```

#### 新增 type 类型
- 扩展 type 支持: `'text' | 'ai-generated' | 'heading' | 'list' | 'code'`

#### 新增链接管理方法
- `addLink(fromBlockId, toBlockId)` - 建立双向链接关系
- `removeLink(fromBlockId, toBlockId)` - 移除链接关系
- `getBlockLinks(blockId)` - 获取 Block 的所有链接（incoming + outgoing）

### 2. TipTap 自定义扩展

#### BlockLink 扩展（双向链接 `[[]]`）
```typescript
export const BlockLink = Node.create({
  name: 'blockLink',
  group: 'inline',
  inline: true,
  atom: true,
  // ...
})
```

**功能**:
- 渲染为可点击的链接样式
- 显示格式: `[[Block标题]]`

- 自动维护双向链接关系

**样式**:

- hover 时下划线
- 圆角边框

#### BlockReference 扩展（块引用 `(())`）

**功能**:

### 4. Editor 组件增强

#### 自动补全检测
```typescript
onUpdate: ({ editor }) => {
  // 检测 [[
```
