## 关键变更说明

### 1. IndexedDB 存储封装 (blockStore.ts)

#### BlockStore 类
```typescript
class BlockStore {
  - init(): 初始化数据库和索引
  - saveBlock(): 保存 Block
  - getBlock(): 获取单个 Block
  - getAllBlocks(): 获取所有 Blocks（按时间倒序）
  - searchBlocks(): 内容搜索
  - getBlocksByTag(): 按标签过滤
  - getAllTags(): 获取所有标签
  - deleteBlock(): 删除 Block
  - updateBlock(): 更新 Block
}
```

#### 数据模型
```typescript
interface Block {
  id: string
  content: string
  type: 'text' | 'ai-generated'
  source: {
    type: 'editor' | 'ai'
    documentId?: string
    aiMessageId?: string
    capturedAt: Date
  }
  metadata: {
    title?: string
    tags: string[]
    createdAt: Date
    updatedAt: Date
  }
}
```

#### 技术实现
- 数据库: `blockos-db`
- Store: `blocks`
- 索引: tags (multiEntry), createdAt, type
- UUID 生成函数
