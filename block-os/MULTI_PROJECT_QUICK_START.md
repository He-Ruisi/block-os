# 多项目工作区快速开始

## 🎉 新功能上线！

BlockOS 现在支持多项目管理、多标签页编辑和灵活的工作区布局！

## 快速体验

### 1. 启动应用

开发服务器已在运行：http://localhost:5173

刷新浏览器页面即可看到新界面！

### 2. 探索新界面

#### 左侧边栏
- 📅 **今日**：默认打开的标签页
- 📁 **项目**：管理你的项目
- **+ 新建项目**：创建新项目

#### 标签栏
- 显示所有打开的标签页
- **+** 按钮：新建标签页
- **⛶** 按钮：全屏模式

#### 编辑器和 AI 面板
- 中间的分隔条可以拖拽调整宽度
- 双击分隔条恢复默认比例

### 3. 创建第一个项目

1. 点击左侧边栏的 **"+ 新建项目"** 按钮
2. 输入项目名称，例如："我的第一个项目"
3. （可选）输入项目描述
4. 点击 **"创建"** 按钮
5. 新项目自动在新标签页中打开！

### 4. 尝试全屏模式

1. 点击标签栏右侧的 **⛶** 按钮
2. 侧边栏和 AI 面板隐藏，编辑器占满整个窗口
3. 再次点击 **⊡** 按钮退出全屏

### 5. 调整布局

1. 将鼠标移到编辑器和 AI 面板之间的分隔条上
2. 分隔条会高亮显示（蓝色）
3. 按住鼠标左键拖拽，实时调整宽度
4. 双击分隔条恢复默认比例（60/40）

### 6. 管理标签页

- **切换标签页**：点击标签页名称
- **关闭标签页**：点击标签页上的 × 按钮
- **新建标签页**：点击 + 按钮

## 核心功能

### ✅ 左侧边栏
- 可收起/展开（点击 ☰ 按钮）
- 显示"今日"和项目列表

- 每个项目独立标签页

- 隐藏侧边栏和 AI 面板
- 编辑器占满整个窗口
- 专注写作

### ✅ 可调整布局
- 拖拽分隔条调整编辑器和 AI 面板宽度
- 双击恢复默认比例
- 实时预览

## 使用技巧

### 1. 项目组织
- 为不同主题创建不同项目
- 例如：工作、学习、个人笔记

### 2. 标签页管理
- 同时打开多个项目进行对比
- 关闭不需要的标签页保持整洁

### 3. 布局调整
- 写作时：编辑器宽一些（70%）
- 与 AI 对话时：AI 面板宽一些（50%）
- 专注写作时：使用全屏模式

### 4. 工作流程
1. 早上打开"今日"标签页，查看今天的任务
2. 创建或切换到相关项目
3. 在项目中写作和整理笔记
4. 需要 AI 帮助时，调整布局显示 AI 面板
5. 完成后关闭标签页
